Tutorial GitHub

  • ls - Listar
  • pwd - Exibe o caminho que você está
  • cd - Sobe uma pasta
  • cd. - Volta uma pasta acima
  • cd ~ - Volta para o diretório raiz
  • git config --global user.name "nome da pessoa" - Configurar a conta OBS: (Usar somente uma vez)
  • git config --global user.email "inserir o email" - Email da conta a ser configurada OBS: (Usar somente uma vez)
  • git init - Avisa o git para ficar de olho na pasta
  • git status - Mostra as modificações feitas nos arquivos
  • git add (arquivo/) - Adiciona o arquivo
  • git add. - adiciona todos os arquivos na pasta
  • git commit -m "nome da alteracao" - etiquetar a pasta
  • git remote add origin LINK - Indicar em qual repositório o projeto será colocado
  • git push origin master - Carrega(puxa) as alterações feitas pelos colegas
  • git clone (endereço do projeto no GitHub) - Copia a pasta de arquivo para a máquina
